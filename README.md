# Orbit CRM

Advanced CRM platform for real estate using Next.js, Supabase, and Gemini AI.

## Features
- Real estate pipeline management
- Lead tracking and analytics
- AI-powered interactions
- Supabase database integration

## Tech Stack
- Next.js 14
- TypeScript
- Supabase
- Tailwind CSS
- Radix UI
- Gemini AI API

## Environment Variables

Create a `.env.local` file with the following variables:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
GEMINI_API_KEY=your_gemini_key
NODE_ENV=production
```

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the application.

## Deployment

This application is deployed on Vercel. Connect your GitHub repository to Vercel for automatic deployments.
